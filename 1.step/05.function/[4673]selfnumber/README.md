# SelfNumber [link](https://www.acmicpc.net/problem/4673)

> d(n)을 n과 n의 각 자리수를 더하는 함수 라고 정의 하였을때, d(n) = m 을 만족하는 n이 존재 하지 않는 수 m 을 셀프넘버라고 할때 10000이하의 모든 셀프넘버를 출력하는 문제

1. d(n) = n + n의 각 자리수의 정의로 부터 d(n) = m에서 m이 n보다 크다는 것을 알 수 있다.

2. 1 부터 자신까지 반복하며 d(n)이 자신과 같은 n이 존재하는지 검사한다.

3. (최적화)**아리스토테네스의 체** 방식을 참고하여 1부터 10000까지 반복하며 d(n)을 체크한다. 자신의 차례까지 체크되지 않은 n의 경우 셀프넘버로 출력한다.
